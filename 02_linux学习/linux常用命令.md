- mkdir 创建文件夹
- touch 创建文件
- pwd 查看路径
- vim fileName 编辑文件
- mv file1 file2 编辑文件名
- cat fileName 打印文件内容
- rm fileName 删除文件
- rm -rf 文件夹名
- cp file1 file2 复制文件
- clear 清屏


## Mac下Terminal指定用Sublime打开文件:
>>`$ ln /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl`<br>
***然后:`subl userName` 即可***
